# PintOS

## Setup
I used the [PintOS distribution here](https://github.com/cksystemsteaching/pintos-for-students/tree/master) as my PintOS version. The Docker container runs on Ubuntu:20.04 with the latest dependencies as outlined in [Intalling PintOS](https://web.stanford.edu/~ouster/cgi-bin/cs140-spring20/pintos/pintos_12.html#SEC163).
